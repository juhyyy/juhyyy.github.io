LLM + RAG Intent Analysis Pipeline

LangChain · LangGraph · FAISS · Pydantic
Problem
  • User logs were noisy, ambiguous, and inconsistently labeled, leading to unreliable intent classification.
Approach
  • Redefined intent categories into meaningful semantic units aligned with user behavior.
  • Built a retrieval pipeline using FAISS + RAG for grounded context retrieval.
  • Introduced Pydantic schema validation to stabilize LLM outputs.
Impact
  • Accuracy improved from 57% → 84%.
  • Reasoning errors reduced by 62.6%.
  • Pipeline stability maintained at 99% across evaluation sets.

Virtual Brain Optimization for Neuromodulation

RL (DDPG) · TensorFlow · OHBM 2024
Problem
  • Personalized brain stimulation required slow, expensive simulation cycles.
Approach
  • Integrated structural & functional data into individualized virtual brain models.
  • Used DDPG reinforcement learning to optimize stimulation parameters.
  • Evaluation based on cosine similarity of expected vs. simulated neural responses.
Impact
  • Simulation accuracy improved by 6.2% on average (up to 15.8%).
  • Runtime improved by 3.6× vs. grid search.
  • Presented at OHBM 2024.
