LLM + RAG Intent Analysis Pipeline

LangChain · LangGraph · FAISS · Pydantic
Problem
  • User logs were noisy, ambiguous, and inconsistently labeled, leading to unreliable intent classification.
Approach
  • Redefined intent categories into meaningful semantic buckets aligned with user behavior.
  • Built a FAISS-backed RAG pipeline to retrieve grounded evidence for each prediction.
  • Enforced response structure using Pydantic schema validation and retry logic in LangGraph.
Impact
  • Accuracy improved from 57% → 84%.
  • Reasoning errors reduced by 62.6%, with RAG-only errors down by 14.1%.
  • Parsing stability reached 99% with zero unsupported, evidence-free answers.

Behavioral Interaction Profiling (ASD)

KMeans · UMAP · LLM Prompt Templates
Problem
  • ASD learner behavior varied widely, making personalized scenario design inconsistent and hard to scale.
Approach
  • Standardized dialogue and behavior logs into comparable metrics (turn-taking, utterance length, question/command ratios).
  • Clustered patterns using KMeans + UMAP and validated internal consistency (e.g., silhouette, DBI).
  • Mapped each cluster to conversational templates and LLM prompt rules for scenario generation.
Impact
  • Produced interpretable profiles with key traits, cautions, and recommended feedback strategies.
  • Formed the core module for an ASD social-skills education concept, including flow diagrams and demo scenarios.

Virtual Brain Optimization for Neuromodulation

RL (DDPG) · TensorFlow · OHBM 2024
Problem
  • Personalized brain stimulation required slow, expensive simulations and manual parameter sweeps.
Approach
  • Integrated fMRI, sMRI, and DTI into individualized virtual brain models.
  • Designed a DDPG reinforcement learning pipeline to search over 6 physiological parameters per node.
  • Evaluated performance via cosine similarity between simulated and real neural signals.
Impact
  • Improved simulation similarity by 6.2% on average (up to 15.8%).
  • Reduced runtime by 3.6× compared to grid search.
  • Results presented as a poster at OHBM 2024.

EEG-based OSA Prediction (XAI)

XGBoost · SVM · SHAP
Problem
  • EEG data was high-dimensional and noisy, making OSA severity and treatment planning difficult to personalize.
Approach
  • Extracted 29 EEG features per patient and standardized channel-level entities.
  • Trained and compared multiple ML models (RF, XGBoost, SVR).
  • Used SHAP to highlight clinically meaningful EEG features and channels.
Impact
  • Achieved 85–88% prediction accuracy for AHI and SpO₂.
  • Proposed a personalized CPAP pressure decision-support flow grounded in interpretable biomarkers.
